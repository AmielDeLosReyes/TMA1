Basic Fundamentals of JavaScript

JavaScript was invented by Brandan Eich in 1995. It was created in Netscape Communications and was designed as a scripting language for the company's flagship, web browser, and Netscape Navigator.

JavaScript:

    Unlike HTML and CSS, JavaScript is a programming language that's used to design webpage behaviour. It handles click events, forms inputs, and other logical functionalities. It helps create an dynamic webpage and makes a website user-friendly.

JavaScript Fundamentals:

  • One of the most basic JavaScript method is the writeln method. This method writes a line of HTML5 text in the HTML5 document.

  • JavaScript also have alert features. It can be used when trying to signal an error or any kind of invalid inputs. The syntax for that is: window.alert("string");.

  • One of the most used method in JavaScript is the .getElementById() method. It grabs an HTML element with the Id specified and we can do other methods on it such as: .innerHTML, .style, and more.

  • There are also String methods that works with strings such as: .length, .substring(), .replace, .toUpperCase(), .concat(), and more. These are important methods when it comes to grabbing String data and also validating them.

  • JavaScript also uses control statements and iterative methods. These are for loops, while loops, do-while loop, if statements, if-else statements, and more. Their main purpose is to provide logic and for evaluations.

  • JavaScript also allows any Mathematical calculations, either for display or for logic. Some of these methods are math.random, math.floor, math.round, and more.

JavaScript Arrays:

  • An array consists of a list of values put into a variable. The correct syntax for it is: const arrayName = [item1, item2, ...]; . Arrays are useful and convenient since it prevents you from writing variables one by one. Instead, we use arrays and iterate through it to either display, find, delete, or add elements/items inside it.

JavaScript Functions:

  • JavaScript allows the creation of functions. The basic syntax for creating a function is: function functionName(param1, param2) { ... } . This allows the programmers to create any function that performs any kind of action or method or calculations.

JavaScript Event Handlers:

  • JavaScript has event handlers that handle certain events that happens to any HTML element. The basic syntax for it is: document.addEventListener("click", function;. The first parameter can include values such as mouseover, focus, keypress, submit, and many more.

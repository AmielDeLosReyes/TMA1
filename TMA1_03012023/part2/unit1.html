Basic Fundamentals of HTML5 and CSS

HTML5 stands for HyperText Markup Language 5. It is a markup language that acts as a skeleton of the website. It contains the structure and the contents of the website.

CSSs stands for Cascading Style Sheets. It is formats the document contents and provides layout, colors, background, fonts, and more. It designs the website and makes it user friendly.

In this tutorial, we will focus on the fundamentals of HTML5 such as tables, forms, page-structure elements, and some new input types. We will also go through the fundamentals of CSS in order for us to design our website.

HTML5:

  • The main elements of HTML5 are html, head, and body. The html element is the parent element of the whole HTML document. The head element contains information about the HTML document such as the title, character set, and some other external links. The body element contains the contents of the website which is displayed on the browser.

  • Most of the elements need start and end tags. End tags is preceded with a forward slash in their brackets. There are also elements that do not need end tags, called void elements. Here's a list of void elements: area , base , br , col , command , embed , hr , img , input , keygen , link , meta , param , source , track , wbr.

  • Importants elements for displaying contents are p, h1 to h6, a (this is called an anchor, it's purpose is to link another page and redirect the user to that linked page when clicked), strong, img, ul and ol (used for lists), table (used to display contents in tabular form), form (used to prompt users to input information, usually for signing up or logging in), input, and more.

HTML5's Page-Structure Elements:

  1. main. The main element is representing the dominant and major content of the body. Developers should put the main contents of their website inside the main element.

  2. article and section. Use the article tag if the content can stand on its own. Use the section tag for subsections only. Contents inside the section can't stand on its own.

  3. aside. The aside element can act as a sidebar of the website. Contents inside this tag can contain background information, quotes, supplementary links, etc.

  4. nav. The nav element is used for navigation. It's purpose is to give access to other links and pages connected to the website. It makes it easier for users to navigate through the website and access other pages.

  5. header and footer. The header element is used for the branding of a website. It can contain the website's title, logo, banners, and also navigation and documentation. The footer element is used to markup contents at the end of each pages. It contains copyright information, navigation links, ownership, etc.

HTML5's Input Types:

  1. date. This input type allows the user to select at date from a dropdown calendar. Its format when saved on the database is yyyy-mm-dd.

  2. email. This input type allows the user to enter an email address. It does not proceed if the text typed in does not contain the "@" and if the text after that is invalid as well.

  3. tel. This input type allows the user to input a telephone number with a specified pattern. Patterns can be specified using "pattern ="" " inside the tel tag.

  4. URL. This input type allows the user to type in a website URL. It validates whether or not the typed in text is a URL or not.

  5. color. This input type allows the user to choose a color from a color slider/picker.

CSS:

  • Inline Style Sheet is declared inside an HTML element. This has the highest prority when it comes to following stylesheets. Inline style is as good practice when it comes to testing a single css edit to an HTML element.
  • Second priority is the Internal Style Sheets. These are declared inside the head element. It's usually used when only one single HTML page has a unique style.
  • The last prioritized style sheet is the External Style Sheet. This is an external css file and is linked by using the "link rel="stylesheet" href="style.css"" syntax. It's used so that most of the pages have a consistent design and style throughout the website.

CSS Fundamentals:

  • CSS can design backgrounds and colors of a specified HTML element. There are the ones used for that: background-image, background-color, background-position, and more.

  • CSS can also alter fonts and formats for texts. There are the ones used: font-size, font-family, font-weight, text-indent, and more.

  • The Box Model is one of the most important concepts when it comes to learning CSS. A developer should learn how to control borders and margins. The whole HTML document can be treated as a Box Model with CSS. These are the ones used for that: border-width, border-style, border-color, margin, margin-top, margin-left, margin-right, float, and more.
